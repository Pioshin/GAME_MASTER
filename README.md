# ☘️ The O'Flannery Mystery — Game Master

Un RPG **Cozy Mystery** a tema irlandese per **2 giocatori**, con un **AI Game Master** alimentato da Google Gemini.

## 🍺 La Storia

La *Pinta d'Oro* — la coppa sacra del festival di San Patrizio — è sparita dal pub "Il Trifoglio Ubriaco"! Seguirete le tracce tra ruscelli dorati, boschi fatati e piazze in festa per scoprire chi l'ha presa... e perché.

## 🎮 Come Giocare

1. Apri `game_master.html` nel browser
2. *(Opzionale)* Inserisci una [Google Gemini API Key](https://aistudio.google.com/apikey) per attivare il Master AI
3. Crea i personaggi con nomi e tratti
4. Esplora le scene, tira la slot machine irlandese 🎰 e interroga i sospettati!

## ✨ Funzionalità

| Feature | Descrizione |
|---|---|
| 🎰 **Slot Machine Irlandese** | Tre rulli con simboli celtico-irlandesi — tre uguali = successo leggendario! |
| 🤖 **AI Game Master** | Chiedi qualsiasi cosa al Master (Gemini 2.0 Flash) con contesto pieno della scena |
| 💾 **Salvataggio Automatico** | La partita si salva in `localStorage` — riprendi da dove eri rimasto |
| 📜 **Diario dell'Avventura** | Log di tutte le azioni, tiri e risposte AI |
| 🖨️ **Schede Stampabili** | Stampa le schede personaggio cartacee |
| 📱 **Responsive** | Funziona su desktop e mobile |

## 🛠️ Requisiti

- Un browser moderno (Chrome, Firefox, Edge, Safari)
- Connessione internet (per Google Fonts, YouTube music, e Gemini AI)
- *(Opzionale)* API Key Gemini per il Master AI
